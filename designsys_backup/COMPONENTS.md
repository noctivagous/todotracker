## Common Usage Patterns

### Basic Component Usage
```html
<!-- Simple button -->
<calcite-button>Click me</calcite-button>

<!-- Button with properties -->
<calcite-button appearance="outline" color="red" disabled>
  Disabled Button
</calcite-button>

<!-- Components with slots -->
<calcite-card>
  <h3 slot="title">Card Title</h3>
  <img slot="thumbnail" src="image.jpg" alt="Card image">
  <p>This is card content.</p>
</calcite-card>
```

### Form Example
```html
<calcite-label>
  Username
  <calcite-input-text name="username" required></calcite-input-text>
</calcite-label>

<calcite-label>
  Password
  <calcite-input-text name="password" type="password" required></calcite-input-text>
</calcite-label>

<calcite-button type="submit">Login</calcite-button>
```

### JavaScript Interaction
```javascript
// Get component reference
const button = document.querySelector('calcite-button');

// Listen to events
button.addEventListener('click', () => {
  console.log('Button clicked!');
});

// Set properties
button.disabled = true;
button.loading = true;

// Call methods
await button.componentOnReady();
```

## Component API Structure

Each component typically has:

- **Properties**: Configuration options (attributes/properties)
- **Events**: Custom events emitted by the component
- **Slots**: Named placeholders for content
- **CSS Variables**: Customizable styling variables
- **Methods**: Programmatic functions

Visit individual component documentation for complete API references.

## Accessibility

All Calcite components are built with accessibility in mind and follow WCAG guidelines. Components automatically provide:
- Proper ARIA attributes
- Keyboard navigation
- Screen reader support
- Focus management
- Color contrast compliance

## Theming and Styling

Components can be customized using:
- **CSS Variables**: Component-specific styling variables
- **Design Tokens**: Global design system variables
- **Mode Classes**: Light/dark theme switching
- **CSS Custom Properties**: Override default styles

See the Foundations guide for detailed theming information.
